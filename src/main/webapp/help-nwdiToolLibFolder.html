Enter the installation folder of your NWDI tools library.

You will find this folder in your NetWeaver Developer Studio 7.0.x installation in the sub folder 'tools'. Copy the whole folder to your Hudson server.

The resulting folder is:

  • /opt/NWDI/tools/lib oder
  • c:\NWDI\tools\lib.

when you copied the contents of the folder to /opt/NWDI/tools.
