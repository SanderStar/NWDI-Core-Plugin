Ein Track in der NWDI kann so konfiguriert werden, dass er mit einem anderen JDK gebaut wird, als dem mit dem die NWDI ausgeführt wird.

In der Konfiguration der NWDI können die Variablen 'JDK_1_3_1_HOME' und 'JDK_1_4_2_HOME' definiert werden, welche auf JDK-Installationen für ältere Java-Versionen verweisen. Diese können benutzt werden, um Tracks für ältere NetWeaver-Installationen zu bauen.

Geben Sie den Pfad zu einer solchen JKD-Installation an, wenn Sie Tracks für NetWeaver 7.0.x bauen wollen. Anderenfalls lassen Sie das Feld leer.
