Geben Sie das Installationsverzeichnis der NWDI-Tools Library an.

Sie finden dieses Verzeichnis unterhalb Ihrer NetWeaver Developer Studio 7.0.x Installation im Verzeichnis 'tools'. Kopieren Sie das komplette Verzeichnis auf den Hudson Server.

Das resultierende Verzeichnis ist dann z.B.:

  • /opt/NWDI/tools/lib oder
  • c:\NWDI\tools\lib.

wenn Sie den Inhalt des Verzeichnisses nach /opt/NWDI/tools kopiert haben.
